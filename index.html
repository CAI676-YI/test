产品总览

PCT_100是一款基于ESP32的化工控制实训装置，集成了传感器采集、继电器控制、OLED显示、WiFi连接和MQTT通信功能。适用于化工自动化教学、实验室实训和智能控制算法研究。

主控芯片

ESP32双核处理器，支持WiFi无线通信

显示系统

OLED显示屏，实时显示温度、光照和控制状态

控制模式

支持手动/自动双模式，灵活切换控制方式

智能感知

DS18B20温度传感器 + 光敏传感器，双重环境监测

核心亮点

实时环境监测

高精度DS18B20温度传感器，精确采集环境温度数据；光敏传感器监测光照强度，可设置触发阈值

双继电器控制

支持照明和风扇控制，可根据温度或光照条件自动切换设备状态

RGB状态指示

多彩LED指示灯，实时反映系统运行状态：正常、告警、WiFi断开、MQTT断开等

云端通信

支持MQTT协议，可接入EMQX云平台，实现远程监控与控制

阈值可配置

温度和光照阈值可通过串口命令动态调整，自动保存到Flash

本地存储

WiFi密码和MQTT配置自动保存到Preferences，开机自动恢复连接

硬件配置

主控芯片 ESP32 (双核 240MHz)
温度传感器 DS18B20 数字温度传感器
光照传感器 光敏电阻模块
显示屏幕 OLED 128x64 (SH1106 I2C)
继电器输出 2路继电器 (照明/风扇)
指示灯 RGB LED (多种状态显示)
按键输入 KEY1 总闸 / KEY2 模式切换
通信接口 WiFi 2.4GHz / MQTT (TCP 8081)

软件系统

基于FreeRTOS实时操作系统，采用多任务架构，主循环、WiFi/MQTT通信、LED状态更新分别运行在独立线程中。

功能模块

模块 功能说明
display OLED显示驱动，实时刷新运行状态
sensor 温度和光照数据采集，阈值判断
relay 继电器控制逻辑，设备开关
rgb_led RGB LED驱动，多状态颜色指示
wifi_config WiFi连接管理，配网模式
mqtt_client MQTT客户端，订阅/发布消息

MQTT通信

支持远程配置和状态查询，可通过MQTT命令动态调整参数。

MQTT命令

mqtt show

查看当前MQTT配置信息

mqtt set broker <地址>

设置MQTT服务器地址

mqtt set port <端口>

设置MQTT服务器端口

mqtt set user <用户名>
mqtt set pass <密码>
mqtt set device <设备ID>

WiFi命令

forget

清除保存的WiFi凭据

reboot

清除保存的WiFi凭据

rainbow

启动彩虹灯效

快速上手

1

硬件连接

连接电源，打开总闸开关(KEY1)，系统启动后RGB灯播放开机动画

2

查看显示

OLED屏幕显示当前温度、光照强度和工作模式

3

按键操作

KEY2长按2秒切换手动/自动模式；手动模式下短按切换继电器状态

4

状态指示

观察RGB灯颜色判断系统状态：绿色正常、黄色温度告警、红色严重告警、蓝色WiFi断开、紫色MQTT断开

5

串口调试

通过串口发送help查看可用命令，支持WiFi和MQTT配置
